Detailed Description

template<typename RangeType, typename BodyType, typename ValueType>
class kernel::IterateTask< RangeType, BodyType, ValueType >

Helper class to be used when an iteration over a range is required. The overriden run() of this class applies BodyType on every item of ValueType that is in the given RangeType
