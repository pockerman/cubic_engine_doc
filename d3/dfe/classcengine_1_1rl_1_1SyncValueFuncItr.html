Detailed Description

template<typename WorldTp>
class cengine::rl::SyncValueFuncItr< WorldTp >

The SyncValueFuncItr class. Models the iterative policy evaluation algorithm for learning a value function \(V\) under a policy \(\pi\). The implementation uses a two array approach. Thus it is assumed the world, action and reward spaces are finite.
