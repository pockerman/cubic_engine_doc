Detailed Description

template<int spacedim>
class kernel::numerics::FaceElement< spacedim, 0 >

A FaceElement<spacedim,0> represents a point in spacedim spatial dimensions. Edges of spatial dimension spacedim return FaceElement<spacedim,0> ptrs when asked about their edges and faces. We do not derive from Element<spacedim,topodim> because we want to have this class to behave like a Point and not like an Element. There is a specialization for 1D that provides some extended functionality like inquiring if the point is on the boundary and on which boundary, which Element owns the point and if it is an internal point who is the neighbor
