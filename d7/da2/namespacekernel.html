Detailed Description

Compute dot product between two vectors

Implements axpy operation on the given vector. Thus, it implements the following update x = a*y op b*z

The vectors x, y, z must all have the same length

Utility class that simply holds the result of a computation along side with a flag that indicates whether the result is valid or not
