Detailed Description

template<typename FunctionTp, typename... InputTp>
class cengine::ElasticNetFunction< FunctionTp, InputTp >

ElasticNet regularization for the function given as a template parameter The workings of the ElasticNet regularization can be read here https://en.wikipedia.org/wiki/Elastic_net_regularization
